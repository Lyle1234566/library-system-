# Salazar Library System Repository

This repository’s application code is under `library system/`.

- Web frontend: `library system/frontend`
- Backend API: `library system/backend`
- Mobile app: `library system/mobile`

Deployment directories:

- Vercel frontend Root Directory: `library system/frontend`
- Render backend Root Directory: `library system/backend`

If Vercel shows a successful deploy with `404: NOT_FOUND`, it is usually deploying the repository root instead of the Next.js app directory.
